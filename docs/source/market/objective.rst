Market Objective
================

Set the welfare-maximization or cost-minimization objective for the market model.

.. autofunction:: flexgridpy.market_models.objectives.init_objective.set_market_objective

The objective is typically set automatically by
:func:`~flexgridpy.market_models.initializations.formulation.formulate_copperplate_market`
and :func:`~flexgridpy.market_models.initializations.formulation.formulate_dc_opf_market`.
